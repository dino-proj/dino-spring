## 🎯 开发主张

- 模块化结构：按照模块划分而非按照Java Class类型划分，同一个功能下的Entity、VO、Service、Repository位于同一目录结构下。

- 非必要不定义接口：非必要情况下，不采用Interface-Implement方式，因为大部分功能模块没有先定义接口，再定义实现的必要，无需浪费开发时间
